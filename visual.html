About this Demo

This interactive visualization demonstrates how different big-data processing systems handle climate data workloads. ClimateSpark introduces native chunking, spatiotemporal indexing, and optimized data locality — allowing it to outperform both Spark and SciSpark, especially for regional climate queries.

Use the dropdown menu to switch between Global and Regional queries, and drag the slider to explore how performance changes as the number of queried states increases.
